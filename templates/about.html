About TigerRides

{% load static %}

TigerRides is a student run app created to streamline the process of students sharing Uber/Lyft/other transportation services to and from the Princeton University campus. This will reduce transportation costs by allowing for easier communication between traveling students. Essentially, we hope to create a ride-share-share application for Princeton students.
